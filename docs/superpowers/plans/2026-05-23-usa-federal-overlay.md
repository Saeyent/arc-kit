### Standard Document Control classification line

```markdown
| Classification | PUBLIC / OFFICIAL / OFFICIAL-SENSITIVE (agencies treating output as CUI: set via `default_classification` user-config) |
```

US federal classification (CUI / Confidential / Secret / Top Secret) is **not** in scope for v1 — the overlay targets civilian unclassified systems. Agencies handling CUI can override the classification line via the standard `default_classification` plugin user-config field.

## Scope

**In scope (v1)**: Federal civilian agencies and the vendors that sell to them.

**Out of scope**: Federal defense (CMMC / RMF / DISA STIGs), state regimes (StateRAMP / TX-RAMP / CJIS), sector-specific (HIPAA / GLBA / SOX / PCI), Section 508 accessibility (deferred to v2). Each is a candidate sibling overlay.
